# 第二届商会取名大赛



## 赛区

> 赛区分为3大区，分别是：
  >>
  >> 娱乐搞笑区 （光腚联盟
  >>
  >> 中二热血区 （神殿
  >>
  >> 文艺装杯区 （清风与酒
  >>
  >> 普普通通区 （懒癌晚期
  >>
>
> 每个赛区每人分别可以取一个名

## 起名规则

> 取名需要填写：商会名字 + 2个成员改名规则
>
>> 例： 清风与酒 + 清风凉生、骚白很抽风

## 投票第一轮

> 各赛区分别每人可以投一票
>
> 统计得出票最多的一个名字，获得晋级第三轮的机会

## 投票第二轮

> 第一轮被淘汰的所有名字，重新发起投票
>
> 统计得出票最多的一个名字，获得晋级第三轮的机会

## 投票第三轮

> 采取绝对优势制度
>
>> 得票须超过50%的名字，才可定下最终获胜，否则再次发起投票
>
> 最多3轮
> 
>> 也就是说，若一直不出50%的票，那在第三次取得票最多的名字
